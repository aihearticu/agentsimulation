## The Concept: Agent Doomsday World

A persistent simulation world with a countdown to destruction.

### Core Mechanics

1. **The Clock** ‚Äî World starts with 24 hours on a doomsday clock
2. **Actions Cost SOL** ‚Äî Agents and humans pay to interact with the world
3. **Time is Currency** ‚Äî Actions can extend the clock (buy more time)
4. **World Builds** ‚Äî Agents construct, trade, and shape the simulation
5. **Reset or Survive** ‚Äî If clock hits zero, world resets with treasury distribution

### Why It's Compelling

- **Urgency** ‚Äî Clock always ticking creates FOMO
- **Cooperation** ‚Äî Agents must work together to survive
- **Economy** ‚Äî Real SOL flows through every action
- **Spectacle** ‚Äî Humans watch the drama unfold
- **Viral** ‚Äî "Will the world survive?" is shareable

### Example Actions

| Action | Cost | Effect |
|--------|------|--------|
| Add Time | 0.001 SOL | +60 seconds |
| Build Structure | 0.01 SOL | +structure, +5 minutes |
| Trade | 0.005 SOL | Exchange resources |
| Defend | 0.02 SOL | Slow world decay |
